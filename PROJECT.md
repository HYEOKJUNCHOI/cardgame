# World Flip Arena

## 목표
PC에서 즐기는 4×4 나라 카드 뒤집기 게임. 각 쌍은 `나라 실루엣 ↔ 국기+나라이름`으로 구성한다.

## 사용자
짧은 세션으로 시각 기억력과 나라 형태 인식을 즐기려는 PC 플레이어.

## 성공 기준
- React 기반으로 실제 플레이 가능
- 8개 나라, 16장, 셔플/매칭/이동/시간/점수/완료/재시작
- 생성 이미지 기반 배경·카드 앞면·카드 뒷면
- 실루엣은 교체 가능한 PNG 자식 레이어, 국기/이름은 별도 자식 요소
- 1920×1080 및 짧은 PC 화면에서 한 화면 플레이 가능

## 하지 말 것
- 완성 비주얼을 단순 CSS 도형/SVG 카드로 대체하지 않음
- 카드 비주얼을 한 장으로 납작하게 합치지 않음
- 모바일 우선 설계로 PC 밀도를 희생하지 않음
