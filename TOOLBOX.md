# TOOLBOX

## 필수
- frontend-design: 비주얼 방향과 스크린샷 자체비판
- universal-ui-linebreak-rules: 제목/버튼/카드 라벨 줄바꿈
- game-ui-gated-asset-layer-workflow: PNG 부모/자식 레이어 계약
- software-delivery-workflows: 빌드·브라우저·클릭 검증

## 이번 프로젝트의 빡센 기준
- 카드 쉘, 뒤, 배경은 실제 생성 PNG
- 실루엣 PNG는 카드 부모에서 독립 교체 가능
- 카드 클릭, 오답 복귀, 정답 유지, 완료 모달, 재시작을 실제 브라우저로 확인
- 1920×1080과 1365×768 스크린샷 확인
- 브라우저 콘솔 오류 0

## 이번에는 쓰지 않음
- SVG 런타임 렌더링
- CSS gradient로 재질 표현
- 상태보드/9090
