# SPEC

## 화면
- 상단 HUD: 타이틀, 이동 수, 시간, 점수, 새 게임
- 중앙: 8열×2행 카드 보드
- 카드 앞면 A: 나라 실루엣 PNG
- 카드 앞면 B: 국기 이모지 + 한글 나라명 + ISO 코드
- 완료: 최종 점수와 다시 도전

## 데이터
대한민국, 일본, 미국, 프랑스, 브라질, 호주, 이집트, 인도.

## 동작
1. 새 덱 셔플
2. 첫 클릭에서 타이머 시작
3. 두 장이 같은 나라이고 서로 다른 카드 타입이면 매칭
4. 오답은 잠시 공개 후 복귀
5. 8쌍 완료 시 타이머 정지 및 결과 표시
6. 새 게임/다시 도전은 모든 상태 초기화

## Acceptance Criteria
- [ ] 16장 렌더링
- [ ] shape/identity 매칭 규칙
- [ ] 중복 클릭과 판정 중 입력 차단
- [ ] 키보드 포커스 표시
- [ ] reduced-motion 대응
- [ ] npm run lint/build 통과
- [ ] 실제 브라우저 클릭 흐름 통과
- [ ] PC 스크린샷 구조/가독성 통과
